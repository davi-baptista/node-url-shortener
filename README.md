# 🚀 URL Shortener API

API de encurtamento de URLs feita com **Node.js**, **TypeScript** e **Prisma**, usando arquitetura limpa e testes automatizados.

## 🔗 Funcionalidades

- Criar URL encurtada
- Buscar URL original

## 🛠️ Tecnologias

- Node.js + TypeScript
- Fastify
- Prisma + PostgreSQL
- Docker
- Vitest
